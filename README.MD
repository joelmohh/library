# Library Managment System
A Node.js application for managing a library's book lending system. 

## Features
- User authentication and authorization
- Book catalog management
- Lending and returning books
- Email notifications for overdue and upcoming due dates
- Admin dashboard for managing users and books
- Scheduled tasks for sending reminders and notifications

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/joelmohh/library
```
2. Navigate to the project directory:
   ```bash
   cd library
   ```
3. Install dependencies:
   ```bash
    npm install
    ```
4. Create a `.env` file in the root directory and add the `.env.example` variables.
5. Start the application:
   ```bash
   npm run dev
   ```
## Usage
- Access the application at `http://localhost:3000`.
- Register a new user or log in with existing credentials.
- Use the admin dashboard to manage books and users.
- The system will automatically send email notifications for overdue books and upcoming due dates.
